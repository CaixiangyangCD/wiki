    流程控制¶

    条件语句 - if¶

    条件语句需要开发者通过指定一个或多个条件，并通过测试条件是否为true来决定是否执行指定语句，并在条件为false的情况下执行另外的语句。

    语法¶

    Go语言中if语句语法如下：

    if 布尔表达式 {
    /* 在布尔表达式为 true 时执行 */
} else {
    /* 在布尔表达式为 false 时执行 */
}

    Warning

    • 布尔表达式不使用括号
    • 不能使用0或者非0表示真假
    • 大括号{}必须存在，即使只有一行代码
    • 左括号必须在if或者else的同一行
    • 在if之后，条件语句之前，可以添加变量初始化语句，使用;进行分隔
    • 不支持三元表达式："a > b ? a : b"

    示例¶

    package main

import "fmt"

func main() {
    // if 语句
    a := 10
    b := 20
    if a > b {
        fmt.Printf("a > b\n")
    } else {
        fmt.Printf("a <= b\n")
    }
}
    执行结果
    a <= b

    Note

    初始变量可以声明在布尔表达式里面，注意作用域

    package main

import "fmt"

func main() {
    // 在布尔表达式里声明变量
    if a := 10; a > 5 {
        fmt.Printf("a > 5\n")
    } else {
        fmt.Printf("a <= 5\n")
    }
}
    执行结果
    a > 5

    条件语句 - switch¶

    switch语句用于基于不同条件执行不同动作，每一个case分支都是唯一的，从上到下逐一匹配。

    Golalng switch分支表达式可以是任意类型，不限于常量。可省略break，默认自动终止。

    语法¶

    switch语法如下：

    switch var1 {
    case val1:
        ...
    case val2:
        ...
    default:
        ...
}

    Note

    变量var1可以是任何类型，而val1和val2则可以使同类型的任意值。类型不被局限于常量或整数，但必须是相同类型；或者最终结果为相同类型的表达式。

    注意

    • 支持多条件匹配
    • 不同的case之间不使用break分隔，默认只会执行一个case
    • 如果要执行多个case，需要使用fallthrough关键字，也可用break终止
    • 分支还可以使用表达式，例如：a > 10

    示例¶

    package main

import "fmt"

func main() {
    // switch语句
    score := 90
    grade := ""

    switch score {
    case 90:
        grade = "A"
    case 80:
        grade = "B"
    case 70, 60:
        grade = "C"
    default:
        grade = "D"
    }

    // case可以是条件表达式
    switch {
    case grade == "A":
        fmt.Printf("优秀!\n")
    case grade == "B":
        fmt.Printf("良好!\n")
    case grade == "C":
        fmt.Printf("中等!\n")
    case grade == "D":
        fmt.Printf("及格!\n")
    default:
        fmt.Printf("不及格!\n")
    }

    fmt.Printf("你的分数是: %v\n你的等级是: %v", score, grade)

}
    执行结果
    优秀!
你的分数是: 90
你的等级是: A

    Note

    fallthrough可以执行满足条件的下一个case

    package main

import "fmt"

func main() {
    // fallthrough语句
    // fallthrough可以添加多个
    a := 100
    switch a {
    case 100:
        fmt.Printf("a = 100\n")
        fallthrough
    case 200:
        fmt.Printf("a = 200\n")
        fallthrough // fallthrough语句会强制执行后面的case语句，不管case表达式是否为true，直到遇到break语句为止，或者是switch语句结束为止
    default:
        fmt.Printf("a = %v\n", a)
    }

}
    执行结果
    a = 100
a = 200
a = 100

    循环语句 - for¶

    Golang中for循环只有for关键字，去除了其它语言中的while和do while

    语法¶

    for init; condition; post {
    /* 循环体语句 */
}

    Note

    init：一般为赋值表达式，给变量赋初值 condition：关系表达式或逻辑表达式，循环控制条件 post：一般为赋值表达式，给变量增量或减量

    for语句执行过程如下： 先对表达式init赋初值 判断赋值表达式init是否满足给定condition条件，若其值为真，满足循环条件，则执行循环体语句，然后执行post，进入第二次循环，再判断 condition；如果condition 的值为假，不满足条件，就终止for循环，执行循环体外语句。

    s := "abc"

for i, n := 0, len(s); i < n; i++ {     // 常见的for循环，支持初始化语句
    fmt.Println(s[i])
}

n := len(s)
for n >  0 {    // 替代 while(n > 0){}
    fmt.Println(s[n])
    n--
}

for {   // 替代 while (true) {}
    fmt.Println(s)
}

    示例¶

    循环输出1到10

    package main

import "fmt"

func main() {
    // 循环输出1-10
    for i := 1; i <= 10; i++ {
        fmt.Printf("i = %v\n", i)
    }
}
    执行结果
    i = 1
i = 2
i = 3
i = 4
i = 5
i = 6
i = 7
i = 8
i = 9
i = 10

    Note

    初始条件和迭代语句都可以省略

    package main

import "fmt"

func main() {
    // 循环输出1-10
    // for i := 1; i <= 10; i++ {
    //  fmt.Printf("i = %v\n", i)
    // }

    i := 1 // 初始化语句
    for i <= 10 {
        fmt.Printf("i = %v\n", i)
        i++ // 迭代语句
    }
}
    执行结果

    range¶

    Golang语言中可以使用for range遍历数组、切片、字符串、map以及通道（channel），通过for range遍历的返回值有如下规律：

    1. 数组、切片、字符串返回索引和值
    2. map返回键和值
    3. 通道只能返回通道内的值
    package main

import "fmt"

func main() {

    // for range
    // 循环数组
    arr := [5]int{1, 2, 3, 4, 5}
    for i, v := range arr { // index、value 都是从复制品中取出。
        if i == 0 { // 在修改前，我们先修改原数组。
            arr[1], arr[2] = 999, 999
            fmt.Println(arr) // 确认修改有效
        }

        arr[i] = v + 100 // 使用复制品中取出的 value 修改原数组。

        fmt.Println(i, v)
    }
    fmt.Println(arr)
}
    执行结果
    [1 999 999 4 5]
0 1
1 2
2 3
3 4
4 5
[101 102 103 104 105]

    数组知识点

    如果指定了数组的长度, 那么就保存了数组的变量，属于值传递，发生赋值行为时会对原数组进行拷贝 如果没有指定数组长度，那么就是切片，属于对数组的引用，内存中只保存了一份，发生赋值行为时，实际上是对数组地址的拷贝，两个变量指向同一个数组

    直接引用类型：

    package main

import "fmt"

func main() {
    arr := []int{1, 2, 3, 4, 5}
    for i, v := range arr {
        if i == 0 {
            arr = arr[:3] // 截断切片，删除索引为 3 的元素。对slice的修改不会影响到range循环
            arr[1] += 100 // 修改索引为 1 的元素。对底层数据的修改会影响到 range 循环。
            fmt.Println(arr)
        }
        fmt.Println(i, v)
    }
    fmt.Println(arr)
}
    执行结果
    [1 102 3]
0 1
1 102
2 3
3 4
4 5
[1 102 3]

    for 和 for range的区别¶

    主要使用场景不同: - for可以遍历array和slice、key为整型递增的map、string - for range可以完成所有for可以做的事情，却能做到for不能做的，包括： - 遍历key为string类型的map并同时获取key和value - 遍历channel

    break关键字¶

    使用break关键字的注意事项：

    1. 单独在select中使用break和不使用break没什么区别
    2. 单独在表达式switch语句，并且没有fallthrough，使用break和不使用break没什么区别
    3. 单独在表达式switch语句，并且有fallthrough，使用break能够终止fallthrough后面的case语句的执行
    4. 带标签的break，可以跳出多层select/switch作用域，让break更加灵活，写法更加简单，不需要使用控制变量一层一层跳出循环，没有带break的只能跳出当前语句块

    跳出for循环¶

    package main

import "fmt"

func main() {
    // break关键字
    for i := 1; i <= 10; i++ {
        if i == 5 {
            break
        }
        fmt.Println(i)
    }
}
    执行结果
    1
2
3
4

    跳出switch循环¶

    package main

import "fmt"

func main() {
    i := 2
    switch i {
    case 1:
        fmt.Println("i = 1")
        break
    case 2:
        fmt.Println("i = 2")
        break
        fallthrough
    case 3:
        fmt.Println("i = 3")
    default:
        fmt.Println("i = default")
        break
    }

}
    执行结果
    i = 2

    跳转到标签¶

    package main

import "fmt"

func main() {
    // break 跳转到标签
MYLABEL:
    for i := 1; i <= 10; i++ {
        for j := 1; j <= 10; j++ {
            if j == 3 {
                break MYLABEL
            }
            fmt.Printf("%v-%v\n", i, j)
        }
    }
    fmt.Println("over...")

}
    执行结果
    1-1
1-2
over...

    goto关键字¶

    goto语句通过标签进行代码间的**无条件跳转**，goto语句可以在快速跳出循环、避免重复退出上有一定帮助。Golang使用goto语句可以简化一些代码的实现过程。例如双层嵌套的for循环要退出时

    跳转到指定标签¶

    package main

import "fmt"

func main() {
    // goto关键字
    a := 1
    if a == 1 {
        goto LABEL1
    } else {
        fmt.Println("a != 1")
    }

    fmt.Println("a = ", a)

LABEL1:
    fmt.Println("a = 1")

}
    执行结果
    a = 1

    跳出多层循环¶

    package main

import "fmt"

func main() {
    // goto跳出多层循环
    for i := 0; i < 5; i++ {
        for j := 0; j < 5; j++ {
            if j == 3 {
                goto breakHere
            }
            fmt.Println("j = ", j, "i = ", i)
        }
    }
breakHere:
    fmt.Println("done")

}
    执行结果
    j =  0 i =  0
j =  1 i =  0
j =  2 i =  0
done
